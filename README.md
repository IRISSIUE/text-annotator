# IRIS Text Annotator

This project allows one to create annotated text using a wide variety of formats for the annotations, and by specifying the text, annotations, and styles in a spreadsheet, instead of needing to learn how to code in HTML/CSS/JS. See this project's [GitHub Pages](#https://irissiue.github.io/text-annotator/) page for an example of what it does.

This project follows the [IRIS Scrolly Story Generator](#https://github.com/IRISSIUE/scrolly-story-generator) pattern of requiring a separate GIT repository and public shared google sheet for each text to annotate. 

As of spring 2026, this is just a test project. If this becomes something used by students in class, this README will be expanded to describe all the setup required to for a student or user to annotate their text.
